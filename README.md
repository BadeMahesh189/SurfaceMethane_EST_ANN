# SurfaceMethane_EST_ANN
Python workflow for estimating daily surface methane (CH4) concentrations across the U.S. using Sentinel-5P TROPOMI, ERA5, and ANN models.



This repository contains the Python workflow used in the peer-reviewed study:

Mahesh Bade., Li, Y. (2026). Mapping U.S. Surface Methane With Artificial Neural 
Networks. Earth Systems and Environment (Accepted). DOI: 10.1007/s41748-025
00989-1.

This code supports reproducibility of the results reported in the paper.
